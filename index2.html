Що таке Linux?

Linux — це сімейство Unix-подібних операційних систем, які використовують ядро ​​Linux розробки фінно-американського програміста Лінуса Торвальдса. ОС, що використовують ядро ​​Linux, називаються дистрибутивами Linux, і є такими ж операційними системами як Microsoft Windows або Apple macOS, але з однією дуже важливою особливістю — їх вихідні коди є відкритими, оскільки вони поширюються під ліцензією GNU GPL, яка передбачає створення вільного та відкритого програмного забезпечення (open source software). Це означає, що будь-який користувач має право вивчати та змінювати вихідний код.

Ядро Linux хоч і відноситься до монолітних ядер, але воно також запозичує і деякі ідеї з мікроядерної архітектури, що означає, що вся операційна система працює в просторі ядра, а драйвера пристроїв (у вигляді модулів) можуть бути легко завантажені (або вивантажені) прямо під час роботи операційної системи.

Дистрибутиви

Arch Linux – він кращий для досвідченіших користувачів, які віддають перевагу максимальній гнучкості та контролю над своєю системою.

CentOS – це безкоштовний та відкритий комплект, що базується на вихідному коді комерційного продукту RHEL.

Debian – це стабільний та універсальний дистрибутив, відомий своєю надійністю.

Ubuntu – це користувальницький дружній варіант, який служить як вхідна точка для багатьох новачків у світ відкритої ОС.

Linux Mint – легкий у використанні набір, створений з акцентом на зручності для користувачів, що переходять з інших ОС.

Manjaro – це дружній комплект, який побудований на Arch Linux.

Команди роботи в Linux CLI

cd – змінити поточну робочу директорію

mkdir – створити нову директорію

head – допомагають виконувати пошук, виводить рядки з початку документа

tail – допомагають виконувати пошук, виводить рядки з кінця документа

touch – дозволяє змінювати різні часові мітки файлу

man – можна отримати довідку і документацію про використання команди

cat – виводить вміст файлу, створює і об'єднує файли'

pwd – виводить робочу директорію

rm – видалити файл або директорію

ls – вивести вміст поточного каталогу

cp – копіювання файлів

BASH - скрипт

#!/bin/bash

user_file="users.txt"

registration() {
    echo "Введіть логін:"
    read name
    echo "Введіть пароль:"
    read password


    if grep -q "^$name:" "$user_file"; then
        echo "Користувач вже існує. Спробуйте інший логін."
    else
        echo "$name:$password" >> "$user_file"
        echo "Реєстрація успішна! Користувача $name додано."
    fi
}

login() {
    echo "Введіть логін"
    read name
    echo "Введіть пароль:"
    read password
    echo  

    if grep -q "^$name:$password$" "$user_file"; then
        echo "Вхід успішний! Ласкаво просимо, $name."
    else
        echo "Неправильно введені дані. Спробуйте ще раз."
    fi
}


echo "Виберіть опцію:"
echo "1. Реєстрація"
echo "2. Авторизація"
read -p "Ваш вибір: " choice

case $choice in
    1)
        registration
        ;;
    2)
        login
        ;;
    *)
        echo "Неправильний вибір. Завершення програми."
        ;;
esac
